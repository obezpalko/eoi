# Lección cinco

[[../40-Deberes/20251013 Lección cinco|Deberes]]

## Información

En esta lección introducimos los **verbos reflexivos** (como *llamarse*), ampliamos nuestra lista de **antónimos** y **sinónimos**, y practicamos vocabulario variado con enfoques fonéticos (letras L, M, N).

---

## Prácticas

### Antónimos (Opuestos)

| Palabra A   | Diferente (!=) | Palabra B |
| :---------- | :------------- | :-------- |
| **Lejos**   | !=             | Cerca     |
| **Lento**   | !=             | Rápido    |
| **Lindo/a** | !=             | Feo/a     |
| **Móvil**   | !=             | Fijo      |
| **Nada**    | !=             | Todo      |
| **Nuevo/a** | !=             | Viejo/a   |
| **Noche**   | !=             | Día       |

### Sinónimos

- **Lista** == Inteligente
- **Nevera** == Frigorífico

### Gramática: Verbos Reflexivos

Los verbos reflexivos indican que la acción recae sobre el mismo sujeto. Se reconocen por el sufijo **-se** en el infinitivo.

#### El verbo "Llamarse"

| Pronombre               | Pronombre Reflexivo | Verbo    |
| :---------------------- | :------------------ | :------- |
| Yo                      | **me**              | llamo    |
| Tú                      | **te**              | llamas   |
| Él / Ella / Usted       | **se**              | llama    |
| Nosotros / as           | **nos**             | llamamos |
| Vosotros / as           | **os**              | llamáis  |
| Ellos / Ellas / Ustedes | **se**              | llaman   |

*Ejemplo de otros verbos similares:* **Bañarse**, **Lavarse**, **Peinarse**.
