# Los Sufijos -ista y -ivo/a

En español, los sufijos cambian el significado de las palabras para referirse a personas o a cualidades de objetos.

## 1. El sufijo -ista (Personas)

Se usa para referirse a profesiones, ocupaciones o personas que practican una actividad.

**Regla de oro:** Es **invariable en género** (no cambia para masculino o femenino). Siempre termina en **-a**.

| Raíz    | Persona (-ista) | Ejemplo                             |
| :------ | :-------------- | :---------------------------------- |
| Deporte | **Deportista**  | *Ella es una deportista famosa.*    |
| Piano   | **Pianista**    | *El pianista toca muy bien.*        |
| Arte    | **Artista**     | *Nosotros somos artistas.*          |
| Taxi    | **Taxista**     | *La taxista conoce bien la ciudad.* |
| Turista | **Turista**     | *Hay muchos turistas en Dénia.*     |

---

## 2. El sufijo -ivo / -iva (Cualidades)

Se usa para crear adjetivos que describen la naturaleza o calidad de algo.

**Regla de oro:** **Concuerda en género** con el sustantivo que describe (-o para masculino, -a para femenino).

| Raíz      | Adjetivo          | Ejemplo                                  |
| :-------- | :---------------- | :--------------------------------------- |
| Deporte   | **Deportivo / a** | *Llevo ropa **deportiva**.*              |
| Educación | **Educativo / a** | *Es un juego **educativo**.*             |
| Actuar    | **Activo / a**    | *Mi abuelo es un hombre muy **activo**.* |
| Crear     | **Creativo / a**  | *Tengo un trabajo **creativo**.*         |
| Fiesta    | **Festivo / a**   | *Hoy es un día **festivo**.*             |

---

## 3. Comparativa: Raíces que comparten ambos

Algunas palabras pueden usar ambos sufijos, pero el significado cambia drásticamente entre la **persona** y la **cualidad**.

| Raíz             | Persona (-ista)  | Cualidad (-ivo/a)  | Ejemplo de diferencia                                            |
| :--------------- | :--------------- | :----------------- | :--------------------------------------------------------------- |
| **Acción**       | **Activista**    | **Activo / a**     | *Es un **activista** muy **activo** en su comunidad.*            |
| **Deporte**      | **Deportista**   | **Deportivo / a**  | *El **deportista** lleva ropa **deportiva**.*                    |
| **Progreso**     | **Progresista**  | **Progresivo / a** | *Ese político **progresista** propone un cambio **progresivo**.* |
| **Colectivo**    | **Colectivista** | **Colectivo / a**  | *El sistema **colectivista** busca el bien **colectivo**.*       |
| **Subjetividad** | **Subjetivista** | **Subjetivo / a**  | *Un filósofo **subjetivista** tiene una visión **subjetiva**.*   |

---

## Resumen de uso

- **¿Eres tú?** → Usa **-ista** (*Yo soy deportista*).
- **¿Es tu ropa, tu coche o tu estilo?** → Usa **-ivo/a** (*Mi coche es deportivo*).
