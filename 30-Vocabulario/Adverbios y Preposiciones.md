# Adverbios y Preposiciones

- **abajo** - down, below / внизу
- **adelante** - forward, ahead / вперед
- **ahí** - there (near) / там (рядом)
- **allí** - there (far) / там (далеко)
- **antes** - before / раньше, перед
- **aquí** - here / здесь
- **arriba** - up, above / вверху, наверху
- **bien** - well / хорошо
- **cerca** - near / близко
- **despacio** - slowly / медленно
- **después** - after / после
- **jamás** - never / никогда (усиленно)
- **lejos** - far / далеко
- **mal** - badly / плохо
- **nunca** - never / никогда
- **siempre** - always / siempre
