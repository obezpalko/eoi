# Palabras interrogativas

## Reglas Generales

1. **Acento diacrítico**: Todas las palabras interrogativas llevan tilde para diferenciarlas de sus homónimos relativos o conjunciones.
2. **Signos de interrogación**: En español es obligatorio usar el signo de apertura (¿) y el de cierre (?).
3. **Preposiciones**: Si el verbo requiere una preposición, esta se coloca siempre delante de la palabra interrogativa (*¿A qué...?, ¿Con quién...?*).

## Lista y Ejemplos

### ¿Qué? (What?)

Se usa para preguntar por una definición, una descripción o una cosa.

- ¿Qué es esto?
- ¿Qué haces los fines de semana?
- ¿Qué libro estás leyendo?

### ¿Quién? / ¿Quiénes? (Who?)

Se usa para preguntar por personas. Tiene forma plural.

- ¿Quién es tu profesor?
- ¿Quiénes son esos chicos?
- ¿Con quién vas a la fiesta?

### ¿Cuál? / ¿Cuáles? (Which? / What?)

Se usa para elegir entre varias opciones o cuando hay un conjunto limitado.

- ¿Cuál es tu color favorito?
- ¿Cuáles de estos zapatos te gustan más?
- ¿Cuál es tu número de teléfono?

### ¿Cómo? (How?)

Se usa para preguntar por la manera, el modo o el estado.

- ¿Cómo estás hoy?
- ¿Cómo se escribe tu apellido?
- ¿Cómo vas al trabajo?

### ¿Cuándo? (When?)

Se usa para preguntar por el tiempo o momento.

- ¿Cuándo es tu cumpleaños?
- ¿Cuándo terminan las clases?

### ¿Dónde? / ¿De dónde? / ¿Adónde? (Where?)

Se usa para preguntar por un lugar, origen o destino.

- ¿Dónde vives?
- ¿De dónde eres? (Origen)
- ¿Adónde vas ahora? (Destino/Movimiento)

### ¿Cuánto/a/os/as? (How much? / How many?)

Varía en género y número según el sustantivo al que acompaña.

- ¿Cuánto cuesta este café?
- ¿Cuánta sal necesitas?
- ¿Cuántos años tienes?
- ¿Cuántas personas hay en clase?

### ¿Por qué? / ¿Para qué? (Why? / What for?)

- ¿Por qué estudias español? (Causa/Motivo)
- ¿Para qué sirve esta herramienta? (Finalidad/Objetivo)

## ¿Qué? vs ¿Cuál?

- **¿Qué? + sustantivo**: ¿Qué película prefieres?
- **¿Cuál? + de + sustantivo**: ¿Cuál de estas películas prefieres?
- **¿Qué? + verbo** (Definiciones): ¿Qué es la libertad?
- **¿Cuál? + verbo** (Elección/Identificación): ¿Cuál es tu dirección? (Elección entre todas las direcciones posibles).
