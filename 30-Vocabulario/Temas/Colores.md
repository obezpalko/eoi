## Notas Gramaticales

### Género de los Colores

Los colores en español son **masculinos** cuando funcionan como sustantivos (el color en sí):
> _Me gusta **el rojo**._

Pero cuando funcionan como **adjetivos**, deben concordar con el sustantivo que describen:
> _La camisa **roja**._ (femenino)
> _El coche **rojo**._ (masculino)

### Colores Invariables

Algunos colores **NO cambian** de género:

- **Naranja**: _el coche naranja / la camisa naranja_
- **Rosa**: _el vestido rosa / la falda rosa_
- **Violeta**: _el bolso violeta / la flor violeta_
- **Turquesa**: _el mar turquesa / la pared turquesa_
- **Beige**: _el sofá beige / la chaqueta beige_
- **Marrón**: _el zapato marrón / la mesa marrón_
- **Gris**: _el gato gris / la nube gris_

### Adjetivos Derivados

- **Rojizo/a**: Que tiene un tono parecido al rojo. (_La piel rojiza_)

## Herramientas

Para descargar imágenes reales basadas en los ejemplos:

```bash
# Instalar biblioteca de búsqueda de imágenes
pip install duckduckgo-search pillow requests

# Ejecutar script de descarga
python3 /home/alexb/src/github.com/obezpalko/eoi/80-Tools/download_color_images.py
```

El script descargará automáticamente imágenes relacionadas con cada ejemplo y las redimensionará a 256x256 píxeles.
