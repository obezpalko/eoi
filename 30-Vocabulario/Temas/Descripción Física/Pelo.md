# Pelo: Colores, Texturas y Peinados

## Colores de Cabello

| **Color de Pelo**         | **Ejemplo Masculino**       | **Ejemplo Femenino**            | **Descripción del Color**                                                                                                                     |
| ------------------------- | --------------------------- | ------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| **Rubio / Rubia**         | _Mi primo es **rubio**._    | _Mi hermana es **rubia**._      | Es el color que varía desde el amarillo muy claro (casi blanco) hasta el dorado intenso.                                                      |
| **Moreno / Morena**       | _El actor es **moreno**._   | _La modelo es **morena**._      | Un color que abarca los tonos oscuros, generalmente castaño oscuro, marrón oscuro o incluso negro. Es el más común en España y Latinoamérica. |
| **Castaño / Castaña**     | _Mi jefe es **castaño**._   | _Mi vecina es **castaña**._     | Un tono de marrón que se asemeja al color de las castañas o al café. Se considera más claro que el moreno.                                    |
| **Pelirrojo / Pelirroja** | _El niño es **pelirrojo**._ | _La cantante es **pelirroja**._ | Color rojizo o anaranjado intenso. Es el color más infrecuente y a menudo está asociado a las pecas.                                          |
| **Canoso / Canosa**       | _Mi abuelo es **canoso**._  | _La señora es **canosa**._      | Describe el pelo que ha perdido su pigmentación original, volviéndose gris o blanco debido a la edad.                                         |
| **Calvo / Calva**         | _El señor es **calvo**._    | _La mujer es **calva**._        | Aunque técnicamente no es un color, describe la ausencia total o parcial de cabello.                                                          |

### Notas de Matices Importantes (Colores)

1. **Uso de Ser vs. Tener**: Generalmente usamos el verbo Ser (ser rubio, ser morena) para describir a la persona de forma general. Sin embargo, también se puede usar Tener con el sustantivo pelo.
     > _Ella tiene el pelo castaño (El adjetivo concuerda en masculino singular con la palabra pelo)._
2. **Diferencia entre Moreno y Negro**: Aunque el pelo puede ser negro, en español es más natural decir que una persona es morena (de pelo oscuro) que decir "es negra" (lo que generalmente describe el color de piel o raza).
3. **Pelo Teñido**: Para describir el pelo que ha sido cambiado artificialmente, usamos el adjetivo teñido/a.
     > _Mi amiga se tiñó y ahora está rubia teñida._

## Texturas, Longitudes y Peinados

| **Tipo / Estilo** | **Ejemplo Masculino**                   | **Ejemplo Femenino**                   | **Descripción en Español**                                             |
| ----------------- | --------------------------------------- | -------------------------------------- | ---------------------------------------------------------------------- |
| **Largo**         | _El músico tiene el pelo **largo**._    | _María tiene el pelo muy **largo**._   | Cabello que tiene mucha longitud (baja de los hombros).                |
| **Corto**         | _Carlos lleva el pelo **corto**._    | _Mi abuela lleva el pelo **corto**._   | Cabello con poca longitud (encima de los hombros).                     |
| **Liso**          | _El niño tiene el pelo **liso**._       | _Ana tiene el pelo **liso**._        | Pelo recto, sin curvas, ni ondas, ni bucles.                           |
| **Rizado**        | _Mi primo tiene el pelo **rizado**._    | _Ella tiene el pelo **rizado**._       | Pelo con bucles, espirales o curvas muy definidas.                     |
| **Ondulado**      | _El actor tiene el pelo **ondulado**._  | _La chica tiene el pelo **ondulado**._ | Un punto medio entre liso y rizado. Tiene forma de olas (como el mar). |
| **Media melena**  | _Él lleva **media melena**._            | _María lleva **media melena**._        | Longitud intermedia, generalmente a la altura de los hombros.          |
| **Rapado**        | _Mi tío tiene el pelo **rapado**._      | _La actriz lleva el pelo **rapado**._  | Pelo cortado extremadamente corto con máquina (casi al cero).          |
| **Suelto**        | _Él lleva el pelo **suelto**._          | _Ana hoy lleva el pelo **suelto**._  | Pelo libre, sin gomas, ni pinzas, ni ataduras.                         |
| **Recogido**      | _Él lleva el pelo **recogido**._        | _Ella lleva un **recogido** elegante._ | Pelo atado o sujetado para que no moleste en la cara.                  |
| **Coleta**        | _El futbolista lleva **coleta**._       | _Ana se ha hecho una **coleta**._    | Pelo recogido hacia atrás con una goma elástica.                       |
| **Trenza(s)**     | _El vikingo lleva una **trenza**._      | _Ana lleva dos **trenzas**._         | Peinado que entrelaza tres mechones de pelo.                           |
| **Moño**          | _El samurái lleva un **moño**._         | _La bailarina lleva un **moño**._      | Pelo recogido en forma de bola redonda (alto o bajo).                  |
| **Flequillo**     | _Él se ha dejado **flequillo**._        | _María tiene **flequillo**._           | Pelo cortado sobre la frente, encima de las cejas.                     |
| **Rastas**        | _El cantante lleva **rastas**._         | _Mi vecina lleva **rastas**._          | Tubos de cabello enredado intencionalmente.                            |
| **Calva (zona)**  | _Él tiene una **calva** pequeña._       | _Ella tiene una **calva** ahí._        | Una pequeña zona de la cabeza donde no hay pelo (por estrés o edad).   |
| **Entradas**      | _Carlos no tiene **entradas**._      | _(Raro en mujeres)_                    | Falta de pelo en las esquinas superiores de la frente.                 |
| **Fino**          | _Tengo el pelo muy **fino**._           | _Ella tiene el pelo **fino**._         | Pelo con poco grosor o volumen (se rompe fácil).                       |
| **Grueso**        | _El chico tiene el pelo **grueso**._    | _La modelo tiene el pelo **grueso**._  | Pelo con mucho volumen, fuerza y cantidad.                             |
| **Encrespado**    | _Con la lluvia se pone **encrespado**._ | _Ella tiene el pelo **encrespado**._   | Pelo rebelde, con electricidad estática o "frizz".                     |
| **Mechas**        | _Él se ha hecho **mechas**._            | _María lleva **mechas** rubias._       | Mechones de pelo teñidos de otro color para dar luz.                   |

### Notas de Matices Importantes (Estilo)

1. **Concordancia con "Pelo" (Muy Importante):**
    - Los adjetivos _largo, corto, liso, rizado, teñido_ describen la palabra **PELO** (masculino singular).
    - No importan el género de la persona.
    - **Incorrecto:** _María tiene el pelo larga._
    - **Correcto:** _María tiene el pelo **largo**._
2. **Verbo TENER vs. LLEVAR:**
    - **Tener:** Características naturales (rizado, liso, fino).
    - **Llevar:** Estilos, cortes o peinados (coleta, trenzas, rapado, suelto).
        > Ana **tiene** el pelo liso, pero hoy lo **lleva** recogido en una coleta.
3. **La palabra "Pelo" vs. "Cabello":**
    - **Pelo:** Es la palabra más común en el día a día en España.
    - **Cabello:** Es más formal, poético o de uso comercial (shampoo commercials).
