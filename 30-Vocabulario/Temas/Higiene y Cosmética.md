# Higiene y Cosmética

Este documento explica palabras relacionadas con el cuidado personal y la belleza, centrándose en las diferencias entre productos similares.

## 1. El Perfume frente a la Colonia

Aunque ambos sirven para oler bien, tienen diferencias importantes en su composición y uso.

| Característica | **El Perfume** | **La Colonia** |
| :--- | :--- | :--- |
| **Concentración** | Alta (15-40% de aceites esenciales). | Baja (2-4% de aceites esenciales). |
| **Duración** | Larga (puede durar todo el día). | Corta (unas 2-3 horas). |
| **Uso** | Ocasiones especiales o uso moderado. | Uso diario, refrescante, después de la ducha. |
| **Intensidad** | Muy fuerte y concentrado. | Suave y ligero. |

### Ejemplos de uso

- *Me he puesto un **perfume** muy caro para la fiesta.*
- *Después de ducharme, me gusta usar una **colonia** fresca.*

---

## 2. Otros productos relacionados

- **El desodorante**: Para evitar el mal olor del sudor.
- **El maquillaje**: Productos para la cara (pintalabios, rímel, etc.).
- **La crema**: Hidratante para la piel (cara, cuerpo, manos).
- **El champú**: Jabón específico para el pelo.
- **El gel de ducha**: Jabón líquido para el cuerpo.
