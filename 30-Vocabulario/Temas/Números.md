# Números y Reglas de Conteo

## 1. Del 0 al 29
Del 0 al 29 se escriben en **una sola palabra**.

| Número | Español                                            | Notas                                                          |
| :----- | :------------------------------------------------- | :------------------------------------------------------------- |
| 0      | **cero**                                           |                                                                |
| 1      | **uno**                                            | Se convierte en **un** ante sustantivo masculino (*un coche*). |
| 2-9    | dos, tres, cuatro, cinco, seis, siete, ocho, nueve |                                                                |
| 10     | **diez**                                           |                                                                |
| 11-15  | **once, doce, trece, catorce, quince**             | Irregulares.                                                   |
| 16-19  | **dieci**séis, **dieci**siete...                   | *diez y seis* -> dieciséis.                                    |
| 20     | **veinte**                                         |                                                                |
| 21-29  | **veinti**uno, **veinti**dós...                    | *veinte y uno* -> veintiuno.                                   |

## 2. Del 30 al 99
A partir del 30, se escriben separados por **"y"**.

- 30: **treinta**
- 31: **treinta y uno**
- 40, 50, 60...: cuarenta, cincuenta, sesenta, setenta, ochenta, noventa.
- 99: **noventa y nueve**

## 3. Las Centenas (100-999)

### Regla del 100
- **Cien**: Se usa cuando es exacto (*100 casas*) o va antes de mil/millones (*cien mil*).
- **Ciento**: Se usa cuando seguido de otros números (*101: ciento uno*, *150: ciento cincuenta*).

### Concordancia de Género (200-900)
Las centenas a partir de 200 tienen género. Deben coincidir con el sustantivo.
- **Masculino**: Doscient**os** euros.
- **Femenino**: Doscient**as** personas.

| Número | Español (Masc / Fem) | Inglés / Ruso                        |
| :----- | :------------------- | :----------------------------------- |
| 200    | doscientos / as      | two hundred / двести                 |
| 300    | trescientos / as     | three hundred / триста               |
| 400    | cuatrocientos / as   | four hundred / четыреста             |
| 500    | **quinientos** / as  | five hundred / пятьсот (Irregular)   |
| 600    | seiscientos / as     | six hundred / шестьсот               |
| 700    | **setecientos** / as | seven hundred / семьсот (Irregular)  |
| 800    | ochocientos / as     | eight hundred / восемьсот            |
| 900    | **novecientos** / as | nine hundred / девятьсот (Irregular) |

## 5. Reglas Culturales y Curiosidades

### Separadores Decimales y de Miles
En España, la escritura de números sigue normas diferentes a las del inglés (UK/US).

| Función                  | En España (ES) | En Inglés (EN) | Ejemplo ES                 | Ejemplo EN                        |
| :----------------------- | :------------- | :------------- | :------------------------- | :-------------------------------- |
| **Miles** (Thousands)    | **Punto** (.)  | Coma (,)       | **1.500** (Mil quinientos) | 1,500 (One thousand five hundred) |
| **Decimales** (Decimals) | **Coma** (,)   | Punto (.)      | **1,5** (Uno coma cinco)   | 1.5 (One point five)              |

**Resumen:** En España, para separar los miles usamos el **punto** (.) y para los decimales la **coma** (,).

### Fechas y Años
Los años **NO** se dividen en dos partes como en inglés ("nineteen eighty"). Se leen como el número completo.
- **1980**: Mil novecientos ochenta (No "diecinueve ochenta").
- **2025**: Dos mil veinticinco.

### Números de Teléfono
En España, los números de teléfono suelen agruparse de **dos en dos**.
- Número: 612 34 56 78
- Se dice: *seis doce, treinta y cuatro, cincuenta y seis, setenta y ocho*.

### El "Un" delante de sustantivos
El número **uno** y los compuestos acabados en uno (21, 31...) pierden la "o" ante un sustantivo masculino.
- Tengo **un** gato (No *uno gato*).
- Tengo **veintiún** años (No *veintiuno años*).
- Tengo **treinta y un** euros.
- **Excepción**: Si el sustantivo es femenino, cambia a "una": *treinta y una personas*.
